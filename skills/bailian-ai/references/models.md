# 可用模型

> 各命令都有合理的默认模型，大多数场景无需手动指定。仅当需要特定能力时通过 `-m` 切换。

## 文本对话 (chat)

| 模型 | 默认 | 特点 | 推荐场景 |
|------|------|------|---------|
| qwen-plus | ✅ | 均衡，速度快，性价比高 | 日常对话、简单任务 |
| qwen-max | | 最强推理能力 | 复杂分析、长文档、数学推理 |
| qwen-turbo | | 速度最快，成本最低 | 简单分类、信息提取、格式转换 |
| qwen-long | | 超长上下文 | 超长文档分析 |

## 模型选择建议

1. **先用默认**：默认模型覆盖 80% 场景
2. **按需升级**：结果不满意时再切换到更强模型（如 qwen-plus → qwen-max）
3. **成本敏感**：简单任务选 turbo 系列
4. **质量优先**：创意/分析任务选 max/plus 系列

> 完整模型列表请查阅 [百炼模型广场](https://bailian.console.aliyun.com/)
